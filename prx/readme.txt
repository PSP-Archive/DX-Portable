prx.vcproj
    これは、アプリケーション ウィザードで生成される VC++ プロジェクトのメインの
    プロジェクト ファイルです。 
    ファイルを生成した Visual C++ のバージョン情報と、アプリケーション 
    ウィザードで選択したプラットフォーム、構成、およびプロジェクトの機能に関する
    情報が記述されています。

このプロジェクトを使用すると、ウィザードで入力したコマンドを Visual Studio から
呼び出して、ビルド、クリーンアップ、リビルドを実行できます。ビルド コマンドは、
nmake または使用しているツールです。

このプロジェクトにはファイルが含まれていないため、ソリューション エクスプローラ
にファイルが表示されません。
